Панель имитирует пленочные цвета на цифровых фотографиях.

Нажав на кнопку "SN/Instruction", вы увидите окно с полем Computer ID и с предложением ввести Serial Number.

1.Вначале в поле ввода серийного номера будет значение идентификатора Computer ID, его удобно копировать оттуда в буфер обмена для отправки мне.

Туда вместо него надо вводить серийный номер, полученный от меня.

2.Затем, нажмите "SN/Instruction" удерживая клавишу Ctrl для записи параматров кистей в пресет.

Если введен ошибочный серийный номер ( или вы нажали кнопку ОК, не заменив Computer ID на серийный номер ), то нажмите на кнопку "SN/Instruction", удерживая клавишу Shift. Повторите ввод серийного номера.
